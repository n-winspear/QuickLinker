How Quick Linker works

The basic functionality of Quick Linker is simple, it creates a shortcut and maps it to a common URL you visit so that you can navigate around quickly and efficiently without clicking through folders upon folders of favourites. I myself use it to switch between the myriad admin panels in my job.

How to use Quick Linker

If your browser supports the chrome omnibox API (which is most browsers) then you're already set to go! Just add your first quick link or use one of our example ones that's already setup. Enter 'gl' (goto link) into the URL bar hit tab (or space) to activate the extension and enter the shortcute (e.g. 'yt') and push enter to go.

How to setup Quick Linker in Arc (and others)

If you're using a browser that doesn't use the chrome omnibox API (i.e. Arc Browser) then you'll need to setup a Site Search shortcut first.

  1. 1 - Start by opening your browser settings and find the 'Site Search' setting (usually where search engines are managed).
  2. 2 - Create a new 'Site Search' and give it a friendly name (e.g. Quick Linker)
  3. 3 - Choose a shortcut to activate the site search (unfortunately you cannot use the default 'gl'. I use 'fl' as it's next door.)
  4. 4 - In the URL search query enter chrome-extension://anmgdgipllkbgkmlkajkidadhpjaanam/search/search.html?q=%s where anmgdgipllkbgkmlkajkidadhpjaanam is the chrome extension ID. If the ID has changed for some reason you can find it in the URL bar of this page.

How Quick Linker came to be

I built V1 of this tool for my own benefit, however since trying out Arc browser I realised it doesn't work with Arc's command bar. I looked back at the original code for a way to make it work with Arc when I realised there were some big things that needed to be fixed and so I rebuilt it (almost) from scratch with a new design and some much needed features (including Arc support).
