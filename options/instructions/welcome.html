This page is a quick rundown on how this tool works and how to setup site search if your browser doesn't use the traditional chrome omnibox API (like Arc browser)

If your browser does support the chrome omnibox API you're already set to go! Just add your first quick link or use one of our example ones that's already setup by entering 'gl' (goto link) into the URL bar hit tab (or space) to activate the extension and enter the shortcute (e.g. 'yt') and push enter to go.

If you're like me and using a browser that doesn't support the chrome omnibox API (a.k.a Arc) then you'll need to do a bit of setup for this to work. Start by opening your browser settings and find the 'Site Search' setting (usually where search engines are managed). Create a new 'Site Search', give it a friendly name (e.g. Quick Linker), choose a shortcut (unfortunately you cannot use the default 'gl', I have opted to us 'fl' as it's next door.) then in the URL search query enter chrome-extension://anmgdgipllkbgkmlkajkidadhpjaanam/search/search.html?q=%s where anmgdgipllkbgkmlkajkidadhpjaanam is the chrome extension ID. If the ID has changed for some reason see this guide to find the current extension ID.
