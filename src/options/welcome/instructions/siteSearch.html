Setup Instructions for Site Search

To set up the site search, follow these steps:

  1. Open Chrome settings.
  2. Go to "Search engine" and then "Manage search engines".
  3. Click "Add" to create a new search engine.
  4. Fill in the fields:
    • Search engine: Quick Linker
    • Keyword: ql
    • URL with %s in place of query: chrome-extension:///src/search/search.html?q=%s
  5. Click "Add" to save the search engine.
